## ✏️ 如何修改数据

### 方法一：直接编辑 data.json（推荐）

1. 用任何文本编辑器（如记事本）打开 `data.json`
2. 修改以下内容：
   - **选手信息**：名字、标签、头像路径、段位文字
   - **联系方式**：QQ号、微信号、二维码图片路径
   - **管理密码**：adminPassword 字段

3. 保存后刷新网页即可看到更新

### 方法二：网页内编辑（无需接触代码）

1. 点击右下角紫色 **"编辑"** 按钮
2. 在打开的编辑面板中直接修改：
   - 选手名称和标签
   - 各游戏段位文字
   - QQ/微信联系方式
3. 点击 **"保存 data.json"** 按钮下载更新后的配置文件

### 方法三：后台管理（高级）

1. 点击页面底部的 **"后台管理端入口"**
2. 输入管理密码（默认：`1234`）
3. 可上传新图片、添加新选手

## 🚀 上传到 GitHub

1. 创建 GitHub 仓库
2. 将整个 `lovely_game_house` 文件夹上传
3. 开启 GitHub Pages（Settings → Pages → 选择 main 分支）
4. 访问 `https://你的用户名.github.io/仓库名/index.html`

## 📝 data.json 字段说明

```json
{
  "players": [           // 选手列表
    {
      "name": "软糖酱",   // 选手昵称
      "tag": "甜心萝莉音 · 全能金牌中单",  // 特色标签
      "avatar": "picture/头像1.jpg",      // 头像图片路径
      "games": {         // 各游戏战绩
        "无畏契约": {
          "highest": { "img": "图片路径", "text": "显示文字" },
          "current": { "img": "图片路径", "text": "显示文字" }
        }
      }
    }
  ],
  "contact": {           // 联系方式
    "qq": { "img": "picture/qq.jpg", "number": "QQ号" },
    "wechat": { "img": "picture/微信.jpg", "id": "微信号" }
  },
  "adminPassword": "1234"  // 管理密码
}
```

## ⚠️ 注意事项

- 图片路径区分大小写
- JSON 文件格式必须正确（可用在线 JSON 验证器检查）
- 修改 data.json 后记得保存
- 如果网页显示空白，可能是 JSON 格式错误
